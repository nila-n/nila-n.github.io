Lab 1: Artemis

The goal of lab 1 was to setup and familiarize myself with the Sparkfun Artemis board, which will be used for the entire duration of the course and handle all robot to robot-master (me!) communication.

Prelab

To start the lab, all I needed to do was install the latest Arduiono IDE, Version 2.2.1. From there, I installed the necessary libraries to use the Artemis board.

Tasks

Task 1: Blink

Following the lab instructions, I first tested the new board by running the Blink.io file.

Task 2: Serial Communication

Next, I tested serial communication using the example file. It will be important for future debugging to be able to use serial communication to send and receive information.

Task 3: Temperature Sensor Test

Next, I tested the onboard temperature sensor. Readings indicated the ambient temperature of the room to be around 32000 units. When I put my hand on the sensor, this increased to about 32500, indicating that the sensor was able to detect the environmental change.

Task 4: Microphone Test

Finally, I tested the onboard microphone. Using an online keyboard, I played different notes to see how well the microphone could detect the different frequencies. A clear different was seen between each note!

Discussion

Ultimately, this lab helped me familiarize myself with the microcontroller we will be using to control our robot. As someone less experienced with CS, I am glad to have had this lab to experiment on a small scale with the Artemis before having to use it on the robot later on.
